ARE YOU FAST ENOUGHT TO WRITE THIS?

This game is made to check your speed of writing. You can choose the difficulty by clicking on:

'Words'(All you write is words)

'Sentences'(All you write is sentenes)

'Words and sentences'(You write both)

There is a timer in the top-left corner, so when it is 0 - you lose.

Try to be as fast as you can, and DON'T forget about dots, spaces, uppercase and lowercase letters!

You must write EXACTLY the same as in example.

Each time you write given word or sentence, the score increases.
