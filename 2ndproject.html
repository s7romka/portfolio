TRY TO GUESS THE NUMBER!

In this game you need to guess a number. You can change the difficulty by choosing the one you want.

If you pass a specific difficulty, you can get a star near the name of the difficulty.

As soon as you click 'start' button, the game will automatically generate a number provided by given range.

You need to guess a number as fast as you can, because the timer is ticking in the top of the page!

If your number is not a number which is given, the program says either it is greater or less.

If your number equals to the given number, the program congratulates you. If it is not, you need to try again.
